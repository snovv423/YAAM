# YAAM

Локальный агрегатор доставки еды для Чечни (Грозный, Аргун, Гудермес, Шали). Прототип клиентского веб-приложения — без сборщиков и фреймворков, чистый HTML/CSS/JS.

## Структура

```
client/           клиентское приложение (то, что открывает пользователь)
  index.html      разметка экранов
  css/style.css   вся стилистика (Liquid Glass, тёмно-зелёный + янтарный)
  js/data.js      демо-данные: рестораны, меню, кандидаты на голосование
  js/app.js       логика: корзина, чекаут, QR-оплата, статусы заказа, голосование
restaurant-bot/   заглушка — бот ресторана ещё не реализован, см. README внутри
docs/             исходные PDF с продуктовыми решениями и вопросом на оценку
```

## Запуск

Статический сайт, бэкенда нет. Открыть `client/index.html` через любой локальный сервер (не `file://` — часть API браузера требует http):

```
cd client && python3 -m http.server 8080
```

Дальше — `http://localhost:8080`.

## Как это соотносится с архивом решений

`docs/YAAM-polnyy-arhiv-v6.pdf` — источник истины по продуктовым решениям (~180 вопросов-ответов). Код в `client/` реализует его не на 100%: в частности, промежуточные статусы «Оплачено, ждём подтверждения» / «Ждём ответа ресторана» были добавлены при аудите 2026-07-05 — до этого в прототипе их не было.
